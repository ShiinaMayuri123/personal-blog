# 个人博客

这是一个基于 Vite + React 的轻量个人博客项目骨架。

## 快速启动

```bash
npm install
npm run dev
```

## 项目结构

- `src/main.jsx` - 应用入口
- `src/App.jsx` - 根路由与页面容器
- `src/pages/Home.jsx` - 文章列表页
- `src/pages/ArticleDetail.jsx` - 文章详情页
- `src/components/ArticleCard.jsx` - 文章卡片组件
- `src/data/articles.js` - 文章内容与元数据
- `src/index.css` - 全局样式

## 扩展方向

- Markdown 内容支持
- 后端 API 或静态站点生成
- 标签、搜索、分页
- CMS 或内容管理功能
